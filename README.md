# Happy - NLW
Repository to keep the code developed at the Next Level Week #3.

## Techs
- [x] NodeJS
- [x] SQLite e TypeORM
- [x] ReactJS
- [x] ReactNative
- [x] Expo
- [x] TypeScript

## Starting Dev Environment
1. Run ```yarn install``` in every folder
2. Run ```yarn dev``` in the backend folder and ```yarn start``` in the mobile/web folder
